📖 项目介绍

这是一套以 AI 开发实战 + 后端架构设计 为核心的项目教程。基于 Spring Boot 3、LangChain4j 和 Vue 3 开发，对标大厂标准的 企业级 AI 代码生成平台。

🚀 四大核心能力

1. 智能代码生成

用户只需输入需求描述，AI 会自动分析并选择最优生成策略。通过工具调用（Function Calling）生成代码文件，并支持 流式输出（Streaming），让用户实时看到 AI 的思考和执行过程。

<img width="100%" alt="智能代码生成演示" src="https://github.com/user-attachments/assets/20e859d7-1851-46bb-aaa4-9238fbf8bfcb" />

2. 可视化编辑

生成的应用将实时展示，支持进入 编辑模式。用户可以自由选择网页元素，通过与 AI 对话快速修改页面样式或逻辑，实现“所见即所得”的开发体验。

<img width="100%" alt="可视化编辑演示" src="https://github.com/user-attachments/assets/f49851a8-eaff-42bb-8532-1ba48bdb53f3" />

3. 一键部署分享

支持将生成的应用一键部署至云端，系统会自动截取应用封面图。部署完成后，用户可获得独立的可访问地址进行分享，同时支持 完整项目源码下载。

<img width="100%" alt="一键部署演示" src="https://github.com/user-attachments/assets/1a22e922-3a0c-4b6f-91a6-8108fb58734c" />

4. 企业级管理

完备的后台管理系统，包括：

用户/应用管理：全生命周期管控。

系统/业务监控：实时掌握系统运行状态。

指标监控：监控 AI 调用情况、Token 消耗及系统性能指标。

❓ 常见问题

1. API 怎么获取？

阿里云百炼：提供免费的新人额度，适合快速上手。

Google AI Studio：目前有较充裕的免费调用额度。

企业直充：直接在各大模型官网（如 OpenAI, Anthropic）充值获取 API Key。

2. 无法访问 Google AI Studio？

如果你在访问相关资源时遇到网络困难，可以参考 GitHub 上热门的开源工具：

Clash Verge Rev: https://github.com/clash-verge-rev/clash-verge-rev

提示：该工具在开发者社区中非常流行，订阅费用通常每月约 20 元左右。
